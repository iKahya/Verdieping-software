# WHAT IS THE GOAL OF THE APP
- This is a Discord selfbot designed to help you clean your DMs quickly and easily. With this application, you can remove your messages from a DM conversation in just a few minutes.
- You can save your account with cleaning your dm's.

## HOW CAN YOU START THE APP

### REQUIRES
- Django, Python and discord.py.
### INSTALLATION & LAUNCH
- If you installed latest versions of this apps open your terminal and get in directory of the app with the followings command cd "your/app/path".
- Start the application: python manage.py runserver
- Once the server starts, Ctrl + click the URL displayed in your terminal to open the application in your browser.
- Congratulations you started the app lets start with how you use it.

## HOW CAN YOU USE IT
- Obtain your Discord token and the user ID of the DM conversation you want to clean.
- Then you can go to service page.
- Enter your Discord token and the target user ID in the provided fields.
- The application will begin deleting your messages from the selected DM conversation.
